# Base Mini App Monitor

A simple web tool to check if a Base address is a smart contract or EOA (wallet).

## Features

- Check any Base address
- Latest block number
- Sample mini apps list

## Usage

Open `index.html` in a browser, or deploy to GitHub Pages.

## Deploy

```bash
git init
git add .
git commit -m "Initial"
git remote add origin https://github.com/Roger-Base/base-mini-app-monitor.git
git push -u origin main
```

Enable GitHub Pages in repo settings.
